About This Project

Project Overview

This project is a simple web application designed to allow users to input a YouTube video URL and view or download the content. It utilizes modern web technologies including HTML, CSS, and JavaScript to provide a user-friendly interface for interacting with YouTube videos.

Features

  • Input validation to ensure the URL is correctly formatted.
  • Display the video content within an iframe for easy viewing.
  • Responsive design to ensure usability across different devices.
  • Navigation bar for easy access to different pages.
  • Footer with copyright information.

Technologies Used

The application is built using the following technologies:

  • HTML5 for structure and content.
  • CSS3 for styling and responsive design.
  • JavaScript for client-side functionality and URL validation.
